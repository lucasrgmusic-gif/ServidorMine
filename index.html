Como se Conectar

A conexão é feita via IPv6. Usamos o DuckDNS para manter o hostname atualizado.

🎮 Versão Bedrock (Mobile / Win10)

Endereço: cronicasminecraft.duckdns.org
Porta: 19132

☕ Versão Java

Muitas vezes é necessário usar o IPv6 direto entre colchetes, pois o Minecraft Java possui dificuldade com hostnames em IPv6. Cole o hostname no NsLookup.io.

Endereço: [2804:30c:917:2e00:...]

🛠️ Script Automatizado (Windows)

Crie um arquivo conectar.bat e cole o código abaixo para copiar o IPv6 do hostname automaticamente:

@echo off
set "dominio=cronicasminecraft.duckdns.org"
for /f "tokens=2 delims=[]" %%a in ('ping -6 -n 1 %dominio% ^| findstr "["') do (
    echo [%%a]| clip
    echo Endereco IPv6 copiado com colchetes: [%%a]
    pause
    exit /b
)
echo Nao foi possivel capturar o IPv6. Verifique a conexao.
pause

Guia de Comandos

Se você nascer em terra alheia e não puder quebrar blocos, use /lands wild, assim você será teleportado ao mapa livre, aonde não foi reivindicado. Também pode fazer uma jornada a pé, até sair do local atual.

Reivindicar terra (Raio de 5 chunks):
/claim radius 5

Renomear sua terra:
/lands rename [Nome]

Configurar o local de Spawn da sua terra:
/lands setspawn

Adicionar amigo a sua terra (Trust):
/lands trust [Nome]

Remover amigo de sua terra (Untrust):
/lands untrust [Nome]

Teleportar para uma terra já reivindicada:
/lands spawn [Nome]

Sair da propriedade de alguém:
/lands leave [Nome]

Deletar sua terra:
/lands delete [Nome]
